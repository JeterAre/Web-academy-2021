What is In line css?


In line css is a way to give HTML attibutes/elements a bit of style. For example with my lab 8, I was able to make a table that allows you to move in between labs and projects with one click. When your cursor hovered over the lab you wanted to go to it would turn a different color. Thats what css could do, give HTML elements some magic and style.


What is internal css?


Internal css is giving the css the room to style your whole page. Internal css is defined in the head and is allowed to be used in all the style elements in the HTML file.


What is External css?


External css is a different file that is made. You have to link the external css file to the HTML file to be able to apply it and us all the commands and designs you can use with css.
